Project Description

Built the "Coastal Threat Alert System", leveraging AI, IoT, and satellite data to deliver real-time disaster alerts and actionable insights for protecting coastal communities and ecosystems.

This project represents a hands-on experience in innovative problem-solving, collaborative development, and climate-tech solutions. The system integrates multiple technologies to provide early warning systems for coastal disasters, helping communities prepare and respond effectively to environmental threats.

Technologies Used

Frontend: React.js, CSS3 Backend: Node.js, Express.js Database: MongoDB APIs: Weather API, Satellite Data API IoT: Raspberry Pi, Sensors Deployment: Heroku, AWS
